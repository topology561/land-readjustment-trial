# §1 街角地三指數選位（手冊規則，W-D.1.3 五段波定案）

## 三指數（唯一評分路徑：`_pk_one_side_v12`）
```
總分 = 0.4×(臨截角長/range截角邊長) + 0.2×(臨側街長/range側街邊長) + 0.4×(真交集/range面積)
```
- **正街角線＝截角斜邊**（視距截角弦），**不是** FRONT_LINE。
- 截角線來源：`_chamfer_line_for_side`（依 `tc['side']` 正典取邊）。

## 街角範圍（range）＝街角最小分配面積
- `_build_corner_range_v2`：block ∩ 條帶{⊥ALLOC 距側街 ≤ T}，**T＝街角退縮＋對應法定最小畸零地寬**（花蓮表）。
- range 面積＝該端「街角最小分配面積」＝G-gate 門檻。診斷「範圍=門檻?」欄必須逐列 ✅。

## 同源原則（分子分母同一顆多邊形）
- **分子**＝parcel∩range 交集多邊形之**邊界**落線長（`_boundary_len_on_line`，容差 ≥1cm）。
- **分母**＝range 多邊形**自身**的截角邊/側街邊——淺 T 時截角會被 ALLOC@T **裁切**（實例：R1左 0m＝4.545 非全長 4.999；R3右 0m＝2.682、3.5m 回 4.939）。裁切是 doctrine，不是異常。
- 同源 ⟹ 項一∈[0,0.4]、項二∈[0,0.2] 由構造成立；clamp 永不觸發（觸發＝多邊形不同源，紅旗）。
- **大地號整包覆蓋 range** ⟹ inter＝range ⟹ 臨長＝range 全邊 ⟹ 拿滿項一項二——這是圖 8 語意的推廣（占據臨街帶者得臨街分），實例 R4 之 628、R1右 之 628。

## Winner 判定（三個陷阱）
1. **winner＝達標中總分最大，非全體最大**——G-gate 先淘汰。實例：R6右 628-38 真交集 129 最大但未達標，winner 是 16.81 的 628-18，**這是對的**；3.5m R1左 628-37 分數最高但卡 G-gate，winner=628-36。
2. 資格＝E-1.7 空間濾網（真交集 > **1.0㎡ 絕對地板**）＋ G估 ≥ 門檻。
3. 全數不合格 → 第 1 宗＝**抵費地（合法輸出、非失敗）**，面積鎖定＝range 面積；池守恆見 g-formula-rules。

## Tiebreaker
總分同分（4dp）→ **重劃前原位次**小者勝。暫行實作＝距角序（角地第 1 宗＝最近角）；**硬 hook：W-D §2 落地時改吃 §2 正典原位次（單一真相源）**——code 有 `TODO(W-D §2)` 標記，grep 得到。
