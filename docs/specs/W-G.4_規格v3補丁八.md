## 一、首宗 W₀＝mp 照實起算（乙定案·正典）

1. **法源錨**（手冊 P93／P111 計算順序 B）：「Ｗ表示分配土地寬度（**宗地側街臨街線實際長度之中點**向宗地分配線作垂直線所量其間之距離）」——W 之量測原點＝**mp**，非 corner。與補丁六 W 正典（mp→逐宗遠側界·垂距）同源一致。
2. **負δ塊首宗 W₀＝(群起點−mp)·â 照實**（可為負；R(W≤0)=0 既有正典）。**禁 clamp 回 corner**。
3. **ΣRw<100% 非 bug**：Rw 係逐宗查表義務（「距側街臨街線 18M(含)以內之分配宗地須計算」）；**法源無 ΣRw 必須閉合 100% 之明文**。ΣRw=100% 僅為三宗填滿 18m 帶之特例（圖證：66.3+17.7+16=100%）；原位次地未填滿帶 → ΣRw<100% 為誠實結果。
4. `app.py:7140-7141` 舊註解（「首筆 W_prev=0 自側街起算·不可自 mp 絕對量」）＝**已廢語意**，施工時刪除或改註引本補丁。

## 二、ΣRw<100% 之去向＝調配池內含吸收（KL 裁＋帳證·正典）

1. **機制**：G 公式 `−Rw·F·l₁` 少減 → 該等宗 G 較大 → 街廓 DXF 守恆下 Σ池較小——**即既有恆等式 `ΔΣ池=−ΔΣG`（閘⑤）**，無另立帳、無地落空。
2. **池不會負（結構性）**：池非「假設 ΣRw=100% 之固定額度」，**池自始＝R街廓 − ΣG(各宗實位實 Rw) 之殘差**——ΣRw<100% 自第一天即內含於池值。後續調配進帶之地**自吃 Rw**（G 自縮）。池量級由 C 費用回收撐持（本案 2,298.80㎡），Rw 楔形差屬二階小量。終驗＝手冊 P111(六)4（抵費地價＋差額地價 對 費用負擔總額）。
3. **帳之層級（覆核紀錄·F.4 3.5m 實測）**：
   - B＝0.171043：一般負擔面積＝共同負擔公設 − 抵充 − **臨街地特別負擔（先自 B 挖除**·`app:1889`／`calc_special_burden_total`＝Σ(正+左+右)×(1−C)·`app:4479`）；特別負擔改由逐宗 `Rw·F·l₁`＋`S·l₂` 收回。
   - C＝0.091319：費用以抵費地回收。
   - 全區恆等：ΣR街廓 22,803.33＝Σ實配 20,504.59＋Σ池 2,298.80（閉合殘差 +0.06·G基準 −1.48＝顯示捨入）；總負擔 Σa−ΣG＝13,920.00（40.44%·Σa=34,423.05）。
4. 前議「缺口轉一般負擔」判讀**作廢**（claude.ai 誤）；「負池 WATCH」**撤銷**。

## 三、â 自動定向（正典·施工規則）

1. **物理義**：â＝垂直 ALLOCLINE 之單位法向；`alloc_normal_axis` 之符號隨 CAD 畫向任意——δ 之正負係 â 符號約定之產物，**非幾何證據**。
2. **定向規則**：â 指向「該組沿臨街**離開街角、進入宗帶累積**」之方向——
   - **主判準**：依管線既有推進方向（右組 `−d_hat`／左組 `+d_hat`·`stepg:577`）定 â 符號；
   - **交叉核**：`(街廓形心 − mp)·â > 0` 獨立驗證；
   - **兩判準不一致 → escalate·禁靜默 fallback**。
3. 同一 R 街廓 ALLOCLINE 單一方向（補丁七 §一.4 重申）；左右組 â 反號屬正常（推進方向相反），定向後 W 語意一致。
