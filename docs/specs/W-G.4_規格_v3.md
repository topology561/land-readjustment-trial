**鐵律層（七級調配成果之硬約束）**

**優先層（非必要，讓位於鐵律）**

**分配層（P.177~179 宗地分配原則，皆「應」）**
- **N0-8**：重劃後土地分配位置**應**按重劃前原有土地相關位次分配於原街廓且面臨原路街線。
- **N0-9**：同一宗跨占街廓分配線兩側，**每側應分配面積已達**原街廓原路街線最小分配面積標準者→**應**於兩側個別分配；**未達**者→**應**向分配面積較大之一側合併分配。
- **N0-10 forced 抵費地＝範圍**：嚴格等於（不足與超額皆違）。
- **N0-13（P.179，同一所有權人數宗土地；皆「應」）**：同一土地所有權人在重劃區內有數宗土地者——
  - 每宗土地應分配之面積**均已達**原街廓原路街線最小分配面積標準者 → **應逐宗個別分配（禁合併、禁移動）**。
  - 其**未達**標準部分 → **應以應分配之面積「較」大者集中合併分配**（＝**A 案型**：同歸戶有達標宗且可分配於原街廓者，未達標宗**一定要**往該等達標宗集中合併；達標宗不只一筆則**可比例合併分配**）。
  - **「較」＝比較級，非最上級**：「較大者」＝該歸戶**達標宗之集合**，非單一最大值。與 §31-1-2 比較級目標集正典同源。
  - **「集中合併分配」之實務解（KL 裁示，鎖定）**：只要未達標之同歸戶土地**往同歸戶土地（達標宗集合）調配**即為合規，**含比例調配**。
    - 範例（KL 原文）：某歸戶重劃前 1/2/3/4 地號分居 R1/R2/R3/R4，G(1)=50、G(2)=60、G(3)=100、G(4)=120，門檻 100 → 1、2 未達標；1、2 往 3、4 集中合併分配，**各自以比例面積調配**至 3 與 4，**均合規**。
  - **本條「應」之真正規範內容＝禁止逸出 A 案型（唯一禁令）**：**同歸戶既有達標宗（可分配於原街廓）者，未達標宗不得彼此「自行集中合併」（例 50+60=110 表面達標）後改走 B 案型（往較淺／後地價較低之街廓分配）**。3、4 存在，1、2 就必須去 3、4。

  - **既有實作已合規**（`F.0_合併決策`：`全達標·無須併→留置原位`＝前段；級0/級0′ 併至標的宗＝後段；併後仍未達標→依「**同歸戶有無達標宗**」分流：`轉F.2（同歸戶R4有達標宗·級1相鄰街廓）`＝A 案型／`轉F.4（逃生門·2×G≥MinA_區·7-5≥½增配）`＝B 案型）。本條之作用為**回歸保護**：本波改 F.3/E1/E2 時不得撞壞此分流判別式。
