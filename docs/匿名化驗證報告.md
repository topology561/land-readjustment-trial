# 地籍資料匿名化驗證報告（UC 花蓮單一7＿德安段）

## 方法：等值保持假名化（equivalence-preserving pseudonymization）
- 全域字典（16 個 PII 欄、同真值→同假值、單射）：統編×4欄、姓名×5欄（含設定義務人）、地址×4欄、出生×4欄。
- 自然人統編(1英9數)→ `A9########`（第2碼9=明顯合成、格式合規）；異常碼→ `Z########`；**8位法人/機關統編及其名稱保留**（公開登記、非個資；可改 MASK_ALL 全匿）。
- 姓名：法人名(與8位統編同列出現過)保留；長度≥5或含團體關鍵字→「測試團體NNN」；餘→同長度假中文名（不與任何真名相同）。
- 地址一律匿（tab1 不使用）；出生非空→ `4NNNNN`；`null` 字面值原樣。
- 非 PII 欄、列序、工作表名、欄頭全數原樣；「重劃區地籍」表無 PII、整表原樣。

## 驗證（複刻 app.py 歸戶指紋 10874-10913 於匿名前後各算一次）
- 宗地 (段,地號) 鍵集：39236 筆，前後一致 ✅
- **歸戶分群劃分同構**：原始 21476 群 == 匿名 21476 群，逐群成員集合完全相同 ✅
- 指紋雙射（原始指紋↔匿名指紋一一對應）✅｜四映射皆單射（不同真值必不同假值）✅
- 同名不同人結構保留：'陳秀娟'(5 統編) → 單一假名＋5 相異假統編 ✅
- app 載入相容：sheet_name='U_LAND'/'重劃區地籍'、header=0 原樣 ✅
