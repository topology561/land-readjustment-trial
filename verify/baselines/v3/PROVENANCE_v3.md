## 財務鎖定形與閉合（run_all 財務閘逐項看守）

```
C = (單位工程費×A_precise + (拆遷費+行政費)×A_precise + 貸款利息) ÷ Σ_blk(後街廓面積×單價)
B = calc_B_value(一般負擔, 重劃前均價, 重劃後均價, A_precise, 已徵收公設)
    一般負擔 = Σ(DXF 共同負擔街廓 area_m2) − 抵充地 − 特別負擔(C)
    特別負擔 = (Σ正面 + Σ側面 負擔面積) × (1 − C)          ← C 耦合 B，故 C 必先算
A = 後街廓單價[所屬街廓] ÷ 重劃前區段單價[重劃前地價區段]     ← **逐宗**
```
